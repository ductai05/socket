1. Cài OpenSSL và path
- Tải Win64 OpenSSL v3.4.0 Light hoặc dùng file có sẵn trong github
  +  https://slproweb.com/products/Win32OpenSSL.html   -> tải Win64 OpenSSL v3.4.0 Light , file EXE -> cài đặt
-  Thêm C:\Program Files\OpenSSL-Win64\bin vào PATH

- Tạo mật khẩu ứng dụng ở https://myaccount.google.com/apppasswords
